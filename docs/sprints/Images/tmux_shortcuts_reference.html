Tmux keyboard shortcuts reference — browse by category using the tabs below.

Prefix key: Ctrl + B
Ctrl+B $Rename current session
Ctrl+B dDetach from session (keep it running)
Ctrl+B sList & switch sessions interactively
Ctrl+B (Switch to previous session
Ctrl+B )Switch to next session
Ctrl+B LSwitch to last (most recent) session
Command line:
tmux new -s nameNew session with name
tmux attach -t nameReattach to a session
tmux lsList all sessions
tmux kill-session -t nameKill a session
Ctrl+B cCreate new window
Ctrl+B ,Rename current window
Ctrl+B &Close current window (asks confirmation)
Ctrl+B nNext window
Ctrl+B pPrevious window
Ctrl+B 0–9Switch to window by number
Ctrl+B wVisual window list & switcher
Ctrl+B fFind window by name
Ctrl+B lToggle to last window
Ctrl+B "Split pane horizontally (top/bottom)
Ctrl+B %Split pane vertically (left/right)
Ctrl+B xClose current pane
Ctrl+B ↑ ↓ ← →Move between panes
Ctrl+B ;Toggle to last active pane
Ctrl+B oCycle through panes
Ctrl+B zZoom/unzoom current pane (fullscreen toggle)
Ctrl+B !Break pane into its own window
Ctrl+B qShow pane numbers (press number to jump)
Ctrl+B SpaceCycle through layout presets
Ctrl+B Ctrl+↑/↓/←/→Resize pane
Ctrl+B {Swap pane with previous
Ctrl+B }Swap pane with next
Ctrl+B [Enter copy mode (scroll / select text)
SpaceStart selection (in copy mode)
EnterCopy selection & exit copy mode
Ctrl+B ]Paste from tmux buffer
q or EscExit copy mode
↑ ↓ / PgUp PgDnScroll in copy mode
/ Search forward in copy mode
?Search backward in copy mode
n / NNext / previous search result

Copy mode uses vi-style keys by default. Add set-window-option -g mode-keys vi to ~/.tmux.conf for full vi bindings.

Ctrl+B ?Show all key bindings (built-in help)
Ctrl+B :Open tmux command prompt
Ctrl+B tShow clock in current pane
Ctrl+B ~Show tmux messages log
Ctrl+B rReload config (if bound in .tmux.conf)
Ctrl+B IInstall plugins (if using tpm)

Tip: Ctrl+B ? always works inside any session to see your current bindings.
